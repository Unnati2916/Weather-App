# React Weather App

A modern and user-friendly **Weather Forecast App** built with **React**, **Tailwind CSS**, and the **WeatherAPI**.  
The app allows users to search for current weather and hourly forecasts by city name or their current location.

---

## Features

- Search weather by city name  
- Get current location weather using Geolocation API  
- Hourly forecast with horizontal scroll  
- Real-time temperature and condition icons  
- Smooth scrolling UI with responsive design  

---

## Tech Stack

- React (Vite)
- Tailwind CSS
- Axios
- React Icons
- WeatherAPI
